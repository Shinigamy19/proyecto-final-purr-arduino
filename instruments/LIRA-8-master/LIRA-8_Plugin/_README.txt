# INSTALLATION INSTRUCTIONS

Thanks for downloading LIRA-8!

1. Please follow these steps for installing Camomile plugins in your respective OS:
https://github.com/pierreguillot/Camomile/wiki/How-to-install-plugins


2. For running the Pd standalone version, please download the latest Pd version and open the _LIRA-8.pd file from the "LIRA-8_Pd_Standalone" folder.
http://msp.ucsd.edu/software.html

# HOW TO PLAY
Use midi notes from C1 to G1 to trigger the sensors or click on them to hold the note.

Use keys from 1 to 8 in the Pd standalone version to trigger the sensors.
